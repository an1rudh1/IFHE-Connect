# IFHE-Connect

IFHE-Connect is a campus-exclusive web application designed for IFHE students to connect through sports activities, cab pooling, and buy & sell listings.

- Sports, Cab Pooling, and Buy & Sell listings

- Email notifications

## Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Authentication: Google OAuth 2.0
- Email: Nodemailer with Gmail SMTP

## Setup Instructions
1. Clone the repository
2. Install dependencies using `npm install`
3. Configure environment variables
4. Run the app using `npm start`

## Note
Login is restricted to IFHE Google email accounts only.
